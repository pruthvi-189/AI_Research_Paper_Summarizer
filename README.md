# AI Research Paper Summarizer & Explainer

This project uses Generative AI and NLP techniques to automatically summarize
research papers and explain complex technical concepts in simple language.

## Features
- Upload research paper PDFs
- Abstract, methodology, results & future scope summarization
- Simplified explanation of technical text
- Interactive Streamlit UI

## Tech Stack
- Python
- Hugging Face Transformers
- Streamlit
- pdfplumber

## Use Case
- Academic research
- MS preparation
- Literature review automation
